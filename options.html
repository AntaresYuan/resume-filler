ApplyMint

把一份简历整理成结构化资料，后面填表时就能一直复用。

多段经历按时间倒序排列（最近的放第一条），填表时会优先使用第一条。字段留空没关系，填表时会自动跳过。
基本信息
姓名、联系方式、个人链接
求职意向
应聘岗位、薪资、到岗时间
教育经历
按时间倒序，最近的学历放第一条
工作经历
按时间倒序，最近的工作放第一条
实习经历
按时间倒序，最近的实习放第一条
项目经验
可选；按时间倒序
技能
按回车或逗号分隔添加
语言
例如：英语 - 流利 / 日语 - N2
自我介绍
会用于匹配 summary / about 这类字段
自定义字段
填表未匹配到的字段，手动保存后可复用
AI 设置
可选：当规则匹配失败时让 LLM 兜底（你自带 API key）
⚠️ 启用后，未匹配字段的标签 + 你的简历会被发送到所选 LLM 服务（你自己的 key）。仅在你信任该提供商时启用。API key 在本地 chrome.storage 用 base64 转码存储——这只能挡偶尔的肉眼浏览，不是加密。
未启用时所有 AI 功能都关闭，简历不会被发送
